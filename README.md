# vite-plugin-civet 

Experimental vite plugin for [civet](https://github.com/DanielXMoore/Civet)

## Installation

```bash
pnpm add vite-plugin-civet -D
```

## Usage

```typescript
import { defineConfig } from 'vite'
import plugin from 'vite-plugin-civet'

export default defineConfig({
  plugins: [plugin()],
})
```
